# 🔍 Just Lost and Found

A comprehensive cross-platform mobile application built to help university students report, track, and return lost items efficiently. 

## ✨ Features
* 🔐 **Authentication:** Secure user login and registration.
* 📢 **Post Items:** Easily report lost or found items with images, descriptions, and specific locations.
* 🔍 **Advanced Search & Filtering:** Filter the feed by category, location, or status (Lost/Found), plus a keyword search.
* 💬 **Real-time Chat:** Communicate with others securely, including the ability to share image attachments.
* 🔔 **Push Notifications:** Automated 5-day reminders for unresolved posts and real-time chat alerts.
* 🌓 **Customization:** Full support for Dark/Light Mode and Arabic/English localization.

## 🛠️ Tech Stack
* **Frontend:** Flutter (Dart)
* **Backend & Database:** Firebase (Authentication, Cloud Firestore)
* **Storage:** Cloudinary API
* **Notifications:** OneSignal API
